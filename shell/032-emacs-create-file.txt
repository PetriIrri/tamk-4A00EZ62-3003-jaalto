How to create a file in emacs:

1. open emacs
2. press C-x C-f
3. emacs will then ask for a file name. Give the desired file name and emacs creates the file.
4. Write some stuff inside the file
5. press C-x C-s to save the written contents
6. finally you can exit with C-x C-c
