Explain used key binding after equal(=) sign:

beginning of line        = 0
end of file              = G
deleting a word          = dw
deleting a line          = dd
deleting a paragraph     = dap
Go to beginning of file  = gg
Go to end of file        = $
Copying line(s)          = yy 
Copying paragraph        = yap 
Turn on/off line numbers = :set number
